# Useful Crates

The following is a list of crates that are used / could be used by
Mallorn.

* sdl2
* tcod-rs (alternative)
* clippy - rust linter
* lazy-static - lazily evaluated static vars
* rand
* strum - macros for working with strings and enums
* error-chain
* bitflags
